\# 🤖 Robot Speed Camera



Мобильное веб-приложение для фиксации превышения скорости вращения роботов через камеру телефона.



\## ✨ Возможности



\- 📸 \*\*Анализ движения в реальном времени\*\* через камеру

\- 🌀 \*\*Расчет скорости вращения\*\* (оборотов в минуту)

\- ⚠️ \*\*Автоматическая фиксация\*\* при превышении 50 об/мин

\- 🖼️ \*\*Сохранение фото нарушения\*\* с точным временем

\- 📜 \*\*История штрафов\*\* с возможностью просмотра и очистки

\- 💾 \*\*Локальное хранение\*\* данных в браузере



\## 🚀 Как использовать



1\. Откройте приложение на телефоне

2\. Разрешите доступ к камере

3\. Наведите камеру на движущийся объект (робота)

4\. При превышении скорости 50 об/мин автоматически создается штраф

5\. Во вкладке "Штрафы" можно просмотреть все нарушения



\## 🛠️ Технологии



\- HTML5 / CSS3

\- JavaScript (ES6+)

\- MediaDevices API (камера)

\- Canvas API (обработка изображений)

\- LocalStorage (хранение данных)



\## 📱 Совместимость



\- ✅ iOS (Safari)

\- ✅ Android (Chrome, Firefox)

\- ✅ Современные браузеры с поддержкой getUserMedia
